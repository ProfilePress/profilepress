Allow registering users to select BuddyPress groups to join.

== Description ==
This extension allows users to select or choose the BuddyPress groups they will like to join at the time of creating an account in your WordPress powered website via your registration form.

== Frequently Asked Questions ==
See the extension page for more info http://profilepress.net/downloads/join-buddypress-groups/
